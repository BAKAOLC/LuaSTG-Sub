# 编译检查单

* 检查exe的资源文件
    * resource.h
    * resource.rc
    * app.ico
    * splash.png

* 检查数据信息
    * 公司名（缩写）
    * 产品名（缩写）
    * log 文件位置

* 检查加密设置
    * 加密的密码
    * 编译时的源代码文件

* 检查特性
    * steam API
        * 启用 steam API
        * steam APP ID
        * 强制从 steam 启动
    * 启用 launch 文件
    * 更多的资源加载log
    * 强制从加密的入口点文件加载
    * 碰撞体缩放

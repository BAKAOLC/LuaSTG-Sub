# LuaSTG (Sub branch)

## Introduce

LuaSTG Sub, is a derivative branch of LuaSTG Plus.  

## Requirements  

1. minimum system version is Windows 7 SP1
2. graphics card must be compatible with Direct3D11  
3. graphics card must at least support D3D_FEATURE_LEVEL_10_0  
4. audio card must be compatible with XAudio2

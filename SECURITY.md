# Security Policy

## Supported Versions

We only make security updates to the latest 2 major versions.

| Version | Status          |
| ------- | ------------------ |
| 0.21.x   | :white_check_mark: Actively developing |
| 0.20.x   | :warning: Security updates only |
| 0.1.x - 0.19.x | :x: EOL |
